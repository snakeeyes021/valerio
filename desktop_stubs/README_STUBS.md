# Desktop File Templates

These `.desktop` files are functional stubs for launching Dorico, the Steinberg Activation Manager (SAM), and the Steinberg Download Assistant (SDA), as well as handling their respective URI schemes (`net-steinberg-sam://` and `net-steinberg-sda://`).
